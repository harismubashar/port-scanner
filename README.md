# port-scanner
Simple port scanner using python

## Usage
Save the script and run the script with the IP address, start port, and end port supplied as arguments:
```bash
./python3 scanner.py <IP address> <port start> <port end>
```
example:
```bash
./python3 scanner.py 192.168.1.1 50 100
```

Keep in mind that your firewall may block the port scans

## Dependencies
* Python 3.x
* No external libraries required
